API RESTful - Rainbow Six Siege

Bienvenido a la API del parcial de Aplicaciones Híbridas.

Endpoints principales

  • /api/usuarios
  • /api/agentes
  • /api/roles

GET

Obtiene información de la colección correspondiente. Ejemplo: `/api/agentes` devuelve todos los agentes, `/api/roles` devuelve todos los roles.

POST

Permite crear un nuevo recurso. Por ejemplo, POST a `/api/roles` crea un nuevo rol, POST a `/api/agentes` crea un nuevo agente asignándole un rol existente.

PUT

Actualiza un recurso existente. Ejemplo: PUT a `/api/agentes/:id` permite modificar el nombre, biografía, país o rol de un agente específico.

DELETE

Elimina un recurso existente. Ejemplo: DELETE a `/api/roles/:id` borra un rol y DELETE a `/api/agentes/:id` borra un agente específico.
